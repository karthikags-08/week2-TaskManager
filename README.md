# 📝 TaskFlow - Interactive Task Manager

A simple and interactive task manager built using **HTML, CSS, and JavaScript**.

---

## 🚀 Features
- ➕ Add tasks
- ✏️ Edit tasks
- ✅ Mark tasks as completed
- ❌ Delete tasks
- 🧹 Clear completed tasks
- 🔍 Filter (All / Active / Completed)
- 🌙 Dark & Light mode

---

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
